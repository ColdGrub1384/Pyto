bookmarks¶

Bookmark files and folders

The bookmark module is a pythonic API for NSURL bookmarks. On iOS and iPadOS, files and folders can often change their path. So hard coding paths isn’t a good idea. Instead, this module saves bookmarks in the disk for selected files. Also, the OS only gives the permission to read and write external files when manually selected. This module takes care of that.

See Accessing external files for more information.

Example:

import pandas as pd
import bookmarks as bm

# Pick a csv file for the first time.
# Then, the previously picked file will be reused.
csv_file = bm.FileBookmark("my_csv_file").path
dt = pd.read_csv(csv_file)
class bookmarks.FileBookmark(name: str = None, path: str = None)¶

A bookmark for storing file paths.

class bookmarks.FolderBookmark(name: str = None, path: str = None)¶

A bookmark for storing folder paths.

class bookmarks.StoredBookmark(name: str = None, path: str = None)¶

A bookmark for storing a file path in the disk.

The bookmark is identified by its name. When a bookmark is initialized with a name for the first time, a file picker will appear.

This is a base class and should not be directly used. Instead, use FileBookmark or FolderBookmark.
