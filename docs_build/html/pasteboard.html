pasteboard¶

Pasteboard Access

This module gives access to the pasteboard.

pasteboard.image() → PIL.Image.Image¶

Returns the image contained in the pasteboard as PIL images.

pasteboard.images() → List[PIL.Image.Image]¶

Returns all images contained in the pasteboard as PIL images.

pasteboard.set_image(image: PIL.Image.Image)¶

Copy the given image to the pasteboard.

Parameters

image – The image to copy.

pasteboard.set_images(array: List[PIL.Image.Image])¶

Copy the given images to the pasteboard.

Parameters

array – A list of images to copy.

pasteboard.set_string(text: str)¶

Copy the given text to the pasteboard.

Parameters

text – The text to copy.

pasteboard.set_strings(array: List[str])¶

Copy the given strings to the pasteboard.

Parameters

array – A list of strings to copy.

pasteboard.set_url(url: str)¶

Copy the given URL to the pasteboard.

Parameters

url – The string to copy as an URL.

pasteboard.set_urls(array: List[str])¶

Copy the given URLs to the pasteboard.

Parameters

array – A list of strings to copy as URLs.

pasteboard.string() → str¶

Returns the text contained in the pasteboard.

pasteboard.strings() → List[str]¶

Returns all strings contained in the pasteboard.

pasteboard.url() → str¶

Returns the URL contained in the pasteboard as a string.

pasteboard.urls() → List[str]¶

Returns all URLs contained in the pasteboard as strings.
